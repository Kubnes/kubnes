Często zadawane pytania

Swój config znajdziesz pod ścieżką:
C:\Program Files (x86)\Steam\steamapps\common\Counter-Strike Global Offensive\game\csgo\cfg
Lub przez Steam'a wchodząc w biblioteke gier następnie klikając prawym przyciskiem myszy na Counter:Strike -> Właściwości -> Zainstalowane pliki -> Przeglądaj. W otwartym ekspolatorze plików należy wejśc do folderu "csgo" następnie "cfg". Nasz config będzie znajdował się pod nazwą "autoexec.cfg"

Aby edytować swój config należy go znaleźć, a następnie wystarczy go otworzyć lub wrzucić już gotowy config o takiej samej nazwie do foldera, w którym znajduje się aktualny config

Tak, gotowy config treningowy oraz casualowy można znaleźć w zakładce "qbnez".
